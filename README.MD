# Embedded-Sensor-CI 🚀

A professional, containerized CI/CD pipeline for embedded C++ applications. This project demonstrates how to build, package, and secure C++ software for Debian-based target systems using modern DevOps practices.

## 🎯 Objective
To transition from manual build processes to a fully automated **Secure Software Development Lifecycle (SecSDLC)**. This project automates the transition from raw C++ source code to a deployable `.deb` artifact using Docker and GitLab CI.

## 🛠 Tech Stack
- **Language:** C++17
- **Build System:** CMake (Out-of-source builds)
- **Packaging:** CPack (Debian `.deb` generation)
- **Environment:** Docker (Immutable build containers)
- **Orchestration:** GitLab CI/CD
- **Quality Assurance:** Static analysis (Cppcheck)
